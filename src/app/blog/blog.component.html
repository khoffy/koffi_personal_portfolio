Spring Boot 3 AWS LAMBDA Serverless Architecture

Generate the project with Maven Archetype: aws-serverless-springboot3-archetype

  • Java version: 17
  • Springboot version: 2.0.0.M1

Adding some necessary dependencies for the project

  • Spring-boot-starter-data-jpai
  • Lombook
  • H2 database

Generating the .zip file to deploy as a serverless application with "mvn clean package" cmd

Deployment on AWS as a LAMBDA FUNCTION>
On AWS console > Lambda function
  • Create a Lambda function
  • Upload the archive through "Upload from" button
  • Then, click on "Save"
  • Then, "Edit" Runtime Settings and the full name of the lambda handler
Expose the LAMBDA FUNCTION via an API GATEWAY
On AWS Console > API GATEWAY
  • Create an API REST Gateway
  • Create a Resource
  • Then, deploy the lambda function
